## Deployed

[WATCH HERE](https://city-info-app.netlify.app/)

## Project description
Cities info app.

Show:
- City name
- Capital common name
- Subregion
- Currencies
- Timezones
- Languages

It is possible to switch the dark/light theme

Used:
- Vue 2.6
- Vue-router
- Vuex
- Scss


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve

```
